Important Facts

  1. Inline display wrap the content of an element tightly around the element's content hence height and width properties of these elements can not be set
  2. Block level elements which have the block display value they take the whole horizontal width of the page and dont allow any elements to appear on either side
  3. Inline-block display combines features of both inline and block elements. Inline-block elements can appear next to each other and we can specify their dimensions using the width and height properties. Images are the best example of default inline-block elements.
